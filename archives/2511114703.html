本地文件与服务器，本地文件与docker镜像的拷贝

发表于2021-10-27|更新于2023-10-27|工具
|字数总计:312|阅读时长:1分钟|阅读量:

将本地路径下的文件上传到服务器内，通过xftp可以实现，安装软件神马的太麻烦，一不小心报的错还是解决不了的，故能少安装一个软件是一个。

将本地文件上传到云服务器：

格式：scp 本地文件路径 user@ip:服务器上文件路径

scp  /home/user/test root@127.0.0.1:/cloud/user/test

将服务器上文件拷贝到本地：

格式：scp user@ip:服务器上文件路径 本地文件路径

scp root@127.0.0.1:/cloud/user/test /home/user/test

在scp的过程中，可能会报如下错:not a regular file

解决办法：添加参数-r

正确命令：scp -r /home/user/test root@127.0.0.1:/cloud/user/test

将本地文件拷贝到docker镜像内：

格式：docker cp 本地路径 容器ID或name:容器目录

docker cp /Users/test/demon 749056ea1637:/opt

将docker内文件拷贝到本地文件夹内：

格式：docker cp 容器ID:本地路径 镜像路径

docker cp 749056ea1637:/Users/test/demon /opt
